  • Using the Ogden material model, the three principal values of the Cauchy stresses can now be computed as
    (3)
    and the principal ﬁrst Piola-Kirchhoﬀ stresses follow by
    $$P_{j} = \lambda_{j}^{-1}\sigma_{j} (4)$$

  • From formula (3)(4)

$$P_{j} =\lambda_{j}^{-1}(-p+\lambda_{j}\frac{\partial{W}}{\partial{\lambda_{j}}})) = -p\lambda_{j}^{-1}+\frac{\partial{W}}{\partial{\lambda_{j}}}(5)$$

  • We use these material parameters :
    $\mu_{1} = 630000, \mu_{2} = 1200,\mu_{3}= -10000$
    $\alpha_{1} = 1.3, \alpha_{2} = 5,\alpha_{3}= -2$
    Strain energy density function
    $$W(\lambda_{1},\lambda_{2},\lambda_{3})=\frac{6300000}{13}\lambda_{1}^{1.3}+\frac{6300000}{13}
    \lambda_{2}^{1.3}+\frac{6300000}{13}\lambda_{3}^{1.3}
    +240\lambda_{1}^{5}+240\lambda_{2}^{5}+240\lambda_{3}^{5}
    +5000\lambda_{1}^{-2}+5000\lambda_{2}^{-2}+5000\lambda_{3}^{-2}
    -\frac{19104360}{13}$$
    $$\frac{\partial{W}}{\partial{\lambda_{1}}} = 630000\lambda_{1}^{0.3}+1200\lambda_{1}^{4}-10000\lambda_{1}^{-3}$$
    $$\frac{\partial{W}}{\partial{\lambda_{2}}} = 630000\lambda_{2}^{0.3}+1200\lambda_{2}^{4}-10000\lambda_{2}^{-3}$$
    $$\frac{\partial{W}}{\partial{\lambda_{3}}} = 630000\lambda_{3}^{0.3}+1200\lambda_{3}^{4}-10000\lambda_{3}^{-3}$$
    let $P_{j} = 0$,$\lambda_{j}$ = 1, $j = 1,2,3$ (undeformed)
    $$p_{j} = \frac{\partial{W}}{\partial{\lambda_{j}}} - P_{j} = 630000+1200-10000 - 0= 621200$$
    The principal first Piola stress $P_{j}$, $j = 1,2,3$
    $$P_{j} = -621200\lambda_{j}+630000\lambda_{j}^{0.3}+1200\lambda_{j}^{4}-10000\lambda_{j}^{-3}$$
